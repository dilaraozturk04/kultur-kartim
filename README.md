# kultur-kartim
Future Talent için geliştirdiğim bu uygulama, dijital dünyada ilk adım bariyerini aşmak için tasarlandı. Yazılımın uçtan uca nasıl "nefes aldığını", verinin işlenip kişiye özel, tıklanabilir bir deneyime dönüşme yolculuğunu öğrendim. Sadece kod yazmayı değil, sosyal bir çekinceyi çözen ürün vizyonu geliştirmeyi bizzat deneyimledim.
